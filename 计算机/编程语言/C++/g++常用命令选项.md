# g++常用命令选项
|  选项      |    解释       |
| ------------- |------------- | 
|    -c    |    只编译          |
|    -E    |    只预编译        |
|    -g    |    生成调试信息        |
|    -o    |    生成指定输出文件        |
|    -O?    |    优化处理        |
|    -shared    |    生成共享目标文件        |
|    -static    |    禁止使用共享连接       |
|    -w    |    不生成任何警告信息        |
|    -Wall    |    生成所有警告信息        |
